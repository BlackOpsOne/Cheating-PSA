# The Solution to Cheating in Zombies

The only solution that will work long term is going to be an anticheat that the community will need to agree on. The anticheat needs to be very robust, you need to check everything, and be smart with it.

The [BO1-AntiCheat](https://github.com/BlackOpsOne/BO1-AntiCheat) project is a great start, it picks up on everything that is known about for Black Ops, but other games have absolutely no means of catching these things.

Also, an anticheat can prevent splicing, it just needs to monitor your game and upload data about it to a remote server and save that data.

## Splicing
This is probably one of the longest standing cheating issues in zombies. The only real way to prove that a game is not spliced, is likely going to be each game being verified with an anticheat or some kind of program that constantly monitors a game, with lots of information about it such as the current round number, when it started/ended, etc. That information should be saved publicly in some way so that the community can go back and tell if a certain game was fully played or not based on the game data collected.
